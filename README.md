# Cognitive assessment (browser)

## Türkçe — Nasıl çalıştırılır?

1. Gerekenleri kur: `requirements.txt` dosyasına bak (Node.js, `npm install`).
2. Proje klasöründe terminal aç.
3. `npm install` sonra `npm start`.
4. Tarayıcıda açılan adresi kullan (genelde `http://localhost:3000`).

---

## English — How to run

1. Install what’s listed in `requirements.txt` (Node.js, then `npm install`).
2. Open a terminal in this project folder.
3. Run `npm install`, then `npm start`.
4. Open the URL shown in the terminal (usually `http://localhost:3000`).
